# Contact Management System

A lightweight, human-crafted command-line interface (CLI) application built in Python for managing business and retail store contacts. It provides full CRUD functionality (Create, Read, Update, Delete) with a clean and interactive text-based user interface.

## Features

- **Detailed Contact Schema**: Store contact details including Store Name, Phone Number, Email, and Physical Address.
- **Add Contact**: Input validation to ensure required fields are not left blank.
- **View Contact List**: Clean, formatted display listing all saved entries with just names and phone numbers for quick scanning.
- **Search Functionality**: A flexible, case-insensitive search mechanism that parses both store names and phone numbers.
- **Update Contact**: Interactive updates that allow you to modify individual fields while leaving others unchanged by simply hitting `Enter`.
- **Safe Delete Option**: Built-in confirmation prompt (`yes`/`no`) to prevent accidental deletions of important entries.
